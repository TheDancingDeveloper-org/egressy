# Egressy security policy

Egressy is an early-stage networking project. Security fixes are accepted for
the latest commit on `main`; no older release line is currently maintained.

Please use GitHub private vulnerability reporting after it is enabled for the
repository. Until then, contact the repository owner privately. Do not open a
public issue containing an exploit, credential, WireGuard profile, token,
webhook destination, or observed public address.

Include the affected version or commit, impact, prerequisites, a reproduction
using fake secrets and generalized addresses, and any proposed mitigation. See
`docs/SECURITY.md` for the product's trust boundaries and supported guarantees.
